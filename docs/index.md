# hpc-job-analyst

**AI-assisted HPC job failure analysis for Slurm-based supercomputers.**

`analyze-job` reads an HPC job stdout file, sends it to a local AI proxy,
and returns a structured analysis explaining why the job failed — along with
concrete next steps for the user and, optionally, a draft help desk ticket
for system administrators.

## Key features

- Identifies root causes: hardware/infrastructure failures vs. software/model errors
- Workflow-aware: understands site-specific workflow conventions via context files
- Generates draft help desk tickets with all information admins need
- Portable: designed for any Slurm-based HPC system, not tied to a single site
- All AI inference stays on-premise via a local proxy; no job data leaves the site

## Quick example

```bash
analyze-job /path/to/myjob.o135797758
analyze-job /path/to/myjob.o135797758 --ticket
```
